## 📌 Project Overview

This project dives into customer-level transaction data to understand how customers interact with brands over time. The goal is to surface actionable insights for targeted marketing strategies by identifying behavioral patterns, loyalty signals, and cross-brand transitions.

---

## 🎯 Objectives

- Estimate **brand switching rates** and **repurchase rates** across customer segments
- Identify **loyalty signals** and cross-brand transition patterns
- Build **interactive dashboards** to communicate insights to business stakeholders
- Develop a **brand recommendation framework** using historical purchase behavior *(in progress)*

---

## 🔍 Key Analysis

- Data cleaning, filtering, and transformation using Excel (VLOOKUP/XLOOKUP, pivot tables)
- A/B testing to validate behavioral hypotheses
- Exploratory Data Analysis (EDA) to identify patterns and outliers
- Brand switching matrix to quantify customer movement between brands
- Repurchase rate calculations segmented by customer cohort

---

## 📊 Dashboards

Interactive Power BI dashboards covering:
- Customer repurchase frequency and trends
- Brand performance comparison
- Loyalty segmentation (new, returning, churned)
- Cross-brand transition flow
